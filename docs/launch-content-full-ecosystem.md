## 2. Reddit Post (r/artificial, r/LocalLLaMA, r/ChatGPT)

**Title:** I built a complete portable AI agent framework using nothing but markdown files — identity, memory, tools, workflows, guardrails, and evaluation. Works with any LLM.

**Body:**

Every AI conversation starts from scratch. Your AI doesn't know your name, your preferences, your work context, or what you discussed yesterday. I got tired of re-explaining myself, so I built something.

**aman** is a complete AI companion framework built entirely on markdown files. No databases, no APIs, no accounts. Just text files you own and control.

It has 6 layers, each solving a different problem:

1. **acore** (identity) — A single file that tells the AI who it is and who you are. Personality, values, communication style, trust dynamics. Your AI reads it and behaves accordingly.

2. **amem** (memory) — An MCP memory server that stores corrections, decisions, and patterns automatically. Your AI remembers what it's learned without you doing anything.

3. **akit** (tools) — A portable toolkit. `akit add github` installs real MCP tools on Claude Code/Cursor/Windsurf. On other platforms, it falls back to guided manual steps. 15 tools in the registry.

4. **aflow** (workflows) — Define multi-step AI workflows in markdown. Code review, bug fixing, feature building, daily standups. The AI follows the steps.

5. **arules** (guardrails) — What the AI won't do. "Never push to main without approval." "Always ask before deleting files." Safety boundaries in a readable file.

6. **aeval** (evaluation) — Track your AI relationship over time. Log sessions, record milestones, monitor trust trajectory. Data-driven improvement.

**One command sets it all up:**

```
npx @aman_asmuei/aman
```

It auto-detects your name (from git), your platform, and your tech stack. Zero questions on most setups.

**The key design principle:** every layer is a markdown file. core.md, kit.md, flow.md, rules.md, eval.md. Your AI reads these files as part of its system prompt. Switch from Claude to ChatGPT? Bring your files. No migration needed.

**Not a developer?** There's a web wizard at https://amanasmuei.github.io/acore/ — generate your identity file in 60 seconds from a browser.

Each package works independently. Use just acore for identity. Add amem for memory. Add akit for tools. Mix and match. Or use `aman` to set up everything at once.

7 packages. 335 tests. MIT licensed. Works with ChatGPT, Claude, Gemini, Cursor, Windsurf, Claude Code, Ollama, or any LLM that accepts a system prompt.
